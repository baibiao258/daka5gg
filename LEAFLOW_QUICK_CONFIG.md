# 🚀 Leaflow快速配置指南

## ⚡ 核心配置（5分钟完成）

### 1️⃣ 创建应用
**镜像地址**:
```
ghcr.io/baibiao258/daka5gg:latest
```

### 2️⃣ 环境变量（必需）
| 变量名 | 值 | 说明 |
|--------|-----|------|
| `CHECKIN_USERNAME` | `你的用户名` | 登录账号 |
| `CHECKIN_PASSWORD` | `你的密码` | 登录密码 |
| `GITHUB_ACTIONS` | `true` | **必须设置** |

### 3️⃣ 资源配置
- **CPU**: `500m` - `1000m`
- **内存**: `1024 MiB` (最少1GB)

---

## 🎯 详细步骤

### 步骤1: 登录Leaflow平台
1. 打开 https://leaflow.com
2. 登录你的账号

### 步骤2: 创建新应用
1. 点击"创建应用"或"新建应用"
2. 选择"容器部署"或"Docker容器"
3. 选择"自定义镜像"

### 步骤3: 填写镜像信息
**镜像地址**:
```
ghcr.io/baibiao258/daka5gg:latest
```

### 步骤4: 添加环境变量
在"环境变量"部分添加：

1. **CHECKIN_USERNAME**
   - 值: 你的登录用户名

2. **CHECKIN_PASSWORD**  
   - 值: 你的登录密码

3. **GITHUB_ACTIONS**
   - 值: `true`

### 步骤5: 设置资源限制
**CPU**: `500m` - `1000m`
**内存**: `1024 MiB` (最少1GB)

### 步骤6: 启动应用
1. 点击"创建"或"部署"
2. 等待容器启动
3. 查看日志确认正常运行

- **08:00** - 自动打卡

**时区**: 北京时间 (Asia/Shanghai)

---

## 🛠️ 故障排除

### ❌ 容器启动失败
1. 检查环境变量是否全部设置
2. 确认用户名密码正确
3. 确认内存≥1024MiB

### ❌ 任务执行失败
1. 查看日志中的错误信息
2. 检查网络连接
3. 确认登录凭据有效

### ❌ 内存不足
**解决**: 将内存限制提升到 1536MiB 或更高

---

## 📋 部署后验证

1. **检查容器状态**: 确保显示"运行中"
2. **查看启动日志**: 确认定时任务配置成功
3. **等待定时执行**: 在指定时间观察日志输出
4. **手动测试**: 如需要可手动触发测试
